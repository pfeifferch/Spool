# Spool
## NFC Spool Reader

- Liest NFC Tags im OpenSpool und FilaMan Format aus. 

- nutzt man die Web NFC API direkt im Browser (hauptsächlich Chrome für Android), indem man ein NDEFReader-Objekt erstellt und dessen scan()-Methode aufruft

- Voraussetzungen: Browser-Unterstützung (z.B. Chrome für Android), Nutzerberechtigung, aktivierte NFC-Funktion auf dem Gerät, Aufruf nach Nutzergeste (Klick/Touch)
  
- bei Selfhosting: SSL/https-Verbindung ist notwendig 

- erstellt mit GPT
